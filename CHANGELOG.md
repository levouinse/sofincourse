## [0.1.0] - 2026-02-25

### Added
- Initial release of SofinCourse platform
- Next.js 14 with App Router
- Firebase Authentication (Google & GitHub OAuth)
- Supabase database integration
- Course catalog with 5 sample courses
- 7 real lessons with YouTube video embeds
- Markdown content rendering with syntax highlighting
- Admin panel for course and lesson management
- Responsive design with GitHub dark theme
- Tor detection middleware
- Docker and Docker Compose configuration
- Row Level Security (RLS) policies
- Security measures (XSS, SQL injection, rate limiting)
- API routes for CRUD operations
- User profile page
- Skill tree placeholder page
- SEO optimization (sitemap, manifest)

### Security
- Input validation with Zod
- XSS prevention with DOMPurify
- SQL injection prevention with parameterized queries
- Rate limiting on API routes
- Path traversal protection
- CSRF protection
- Secure authentication flow

### Documentation
- README.md with setup instructions
- DEPLOYMENT.md for production deployment
- CONTRIBUTING.md for contributors
- SECURITY.md for security policy
- LICENSE (MIT)
- .env.example for environment variables

### Courses Included
1. Introduction to Security (2 lessons)
2. Python Basics (2 lessons)
3. JavaScript Fundamentals (1 lesson)
4. Web Security Essentials (1 lesson)
5. Linux Basics (1 lesson)
